# AI Study Assistant

An AI-powered web application that helps students instantly understand any topic and generate practice questions.

## 🚀 Features
- Topic-based explanation
- Practice question generation
- Clean and responsive UI
- Fallback logic for uninterrupted user experience

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 💡 Purpose
This project was built to explore how AI can reduce friction in learning by providing instant value to users without heavy onboarding.
